# ♟️ Lógica de Xadrez em C

Projeto focado na implementação da lógica do jogo de xadrez.

## Recursos implementados

- Estrutura do tabuleiro
- Controle de turno
- Validação de movimentos
- Verificação de caminho livre
- Sistema de captura
- Alternância de jogador

## Compilar

```bash
gcc main.c game.c -o xadrez
